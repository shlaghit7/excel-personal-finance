# 💰 Personal Finance & Expense Management Dashboard (Excel)

## 📌 Project Overview

This project is an Excel-based Personal Finance Dashboard designed to track income, expenses, savings, and budget performance. It provides a clear and interactive way to understand financial behavior and make better financial decisions.

## 🎯 Objective

To build a professional, real-world financial dashboard using Excel that helps users:

* Track income and expenses
* Monitor savings
* Compare budget vs actual spending
* Identify spending patterns

---

## 🛠️ Tools & Skills Used

* Microsoft Excel
* Pivot Tables
* Data Analysis
* Data Cleaning
* Conditional Formatting
* Financial Analysis

---

## 📊 Dataset Description

The dataset includes:

* Date & Month
* Income Sources (Salary, Freelance, etc.)
* Expense Categories (Food, Rent, Travel, etc.)
* Payment Modes (UPI, Cash, Card, Bank Transfer)
* Amount (Income/Expense)

---

## 📈 Key Features

* KPI Cards (Income, Expense, Savings)
* Expense Analysis by Category
* Income vs Expense Trend
* Conditional Formatting for Overspending
* Clean and Professional Dashboard Layout

---

## 🔍 Key Insights

* Identifies top spending categories
* Tracks savings trends over time
* Helps detect overspending
* Improves financial decision-making
